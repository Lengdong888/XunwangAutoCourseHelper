# XunwangAutoCourseHelper
这是一款适配 Tampermonkey、专为武汉理工大学继续教育平台打造的网课辅助脚本，可全自动完成课程学习流程。脚本能够自动识别平台弹出的答题窗口，自主勾选答案、提交作答并关闭解析弹窗；视频播放完毕后，会依据课程目录自动跳转至下一小节，支持同章节连续学习与跨章节切换，全程无需人工点击操作。 核心功能 自动处理课程随机答题弹窗； 一键选答、提交并关闭答案解析页面； 视频结束自动切换下一节，支持跨章节连续学习； 整套学习流程自动化，减少手动操作。 使用须知 平台播放器存在后台限制，浏览器切后台、页面最小化时，播放计时会暂停、进度锁定，脚本无法后台挂机，必须保持课程页面前台可见，该约束由网站播放器与浏览器资源规则决定，和脚本本身无关。 本脚本仅用于脚本开发学习、简化网课重复操作。
